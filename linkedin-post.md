# 🚀 Lorekeeper: Mi nuevo proyecto full-stack

Acabo de lanzar **Lorekeeper**, una plataforma para crear story bibles para escritores, creadores de contenido y equipos de producción audiovisual.

## 🛠️ Tech Stack

- **Backend**: .NET 8 (Auth + World services), FastAPI + Python (AI service)
- **Frontend**: React + Redux + TypeScript
- **Base de datos**: PostgreSQL
- **Despliegue**: Docker + Fly.io (microservicios)
- **API Gateway**: YARP
- **AI**: Groq para generación asistida en español

## ✨ Lo que hace único a Lorekeeper

- Gestión completa de mundos, personajes, localizaciones, facciones y lore
- Generación IA en español (primera lengua)
- Sistema de autenticación robusto con JWT
- Arquitectura de microservicios lista para escalar
- UI/UX diseñada para recruiters que buscan proyectos complejos

## 🎯 ¿Por qué esto importa?

Porque demuestra experiencia real en:
- Arquitectura de microservicios
- Integración con APIs de IA
- Full-stack con múltiples tecnologías
- Despliegue en producción con Docker
- Patrones de diseño RESTful
